# AquaGuard AI Frontend

React + Vite dashboard for AquaGuard AI.

## Run locally

```bash
npm install
npm run dev
```

## Optional live backend

Create `.env.local` from `.env.example` and set:

```env
VITE_API_URL=https://your-ngrok-url.ngrok-free.app
```

If no API URL is provided, the dashboard automatically runs in offline simulation mode.
